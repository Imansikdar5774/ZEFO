Documentation

1. Introduction

DearScript is a fast, beginner-friendly programming language. It removes the strict need for brackets to output data, making coding faster, cleaner, and less confusing than Python or JavaScript.

2. Output Syntax (bik)

Use the bik keyword to print text or math. Spaces and brackets like (), {}, or [] are highly flexible and optional!

bik "Hello World"
bik("Hello World")
bik ("Hello World") // Space allowed!
bik 10 + 5 * 2

3. Comments

Use comments to write notes in your code. The engine will ignore them.

// Single line comment
# Another single line comment

"""
Multi-line comment
"""

4. Variables, Booleans & Arrays (mi)

Use the mi keyword to create variables. A semicolon (;) at the end is strictly required! You can perform math directly inside variables, store Arrays (Lists), and use true / false logic.

mi x = 90;
mi y = 80;
mi isReady = true;
mi arr = [1, 2.5, "hello", 4];
bik (arr)

5. Conditions (if / else)

You can execute code conditionally using if / else: based entirely on Python-like indentation.

mi x = 7;
mi y = 8;

if x > y:
    bik "hello"
else:
    bik "not miss"

6. Data Types

DearScript supports multiple data types effortlessly:

  • str: Text inside quotes, e.g., "Hello"
  • int: Whole numbers, e.g., 90
  • float: Decimal numbers, e.g., 99.5
  • array: Lists of mixed data, e.g., [1, "hello", 3.5]
  • boolean: Logical states, e.g., true or false

7. Data Type Checkers

You can check the type of any variable using dety() or type() directly inside the bik command.

mi x = 90;
bik (dety(x)) // Output: integer
bik (type(x)) // Output: class: int

8. File System & Compilation

DearScript comes with its own Virtual OS. You can save, open, and compile files from the terminal or the new File Manager.

  • ls - View saved files
  • ds filename.ds - Execute a saved file
  • ds filename.ds convert js - Compiles to JS and prompts download!
