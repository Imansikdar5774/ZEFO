Documentation

1. Introduction

DearScript is a fast, beginner-friendly programming language. It removes the strict need for brackets to output data, making coding faster, cleaner, and less confusing than Python or JavaScript.

2. Output Syntax (bik)

Use the bik keyword to print text or math. Spaces and brackets like (), {}, or [] are highly flexible and optional!

bik "Hello World"
bik("Hello World")
bik ("Hello World") // Space allowed!
bik 10 + 5 * 2

3. Comments

Use comments to write notes in your code. The engine will ignore them.

// Single line comment
# Another single line comment

"""
Multi-line comment
"""

4. Variables & Arrays (mi)

Use the mi keyword to create variables. A semicolon (;) at the end is strictly required! You can perform math directly inside variables and store Arrays (Lists).

mi x = 90;
mi y = 80;
mi z = x + y;

// Storing an Array
mi arr = [1, 2.5, "hello", 4];
bik (arr)

5. Data Types

DearScript supports multiple data types effortlessly:

  • str: Text inside quotes, e.g., "Hello"
  • int: Whole numbers, e.g., 90
  • float: Decimal numbers, e.g., 99.5
  • array: Lists of mixed data, e.g., [1, "hello", 3.5]

6. Data Type Checkers

You can check the type of any variable using dety() or type() directly inside the bik command.

mi x = 90;
bik (dety(x)) // Output: integer
bik (type(x)) // Output: class: int

7. Terminal & File System

DearScript comes with its own Virtual OS. Click the 3-dot menu -> Save File. Then open Terminal to manage your files!

  • ls - View saved files in storage
  • ds filename.ds - Execute a saved file
  • clear - Clear terminal screen

8. Compiler (Convert to JS)

NEW! Convert your DearScript files into native JavaScript code and download them automatically.

  • ds filename.ds convert js - Converts bik to console.log.
  • ds filename.ds convert js dom - Converts bik to document.write.
